# IAT Cianorte Presentation

## 📋 Sobre o Projeto

O **IAT Cianorte Presentation** é um sistema web desenvolvido para o **Escritório Regional do Instituto Água e Terra (IAT) de Cianorte**, com o objetivo de centralizar e visualizar dados cruciais para a gestão. O projeto transforma dados brutos em informações visuais claras, permitindo uma análise rápida do desempenho do escritório.

## 📈 Funcionalidades e Recursos

Este projeto simplifica a gestão de dados operacionais com recursos essenciais:
- **Dashboard Interativo:** Visualize em tempo real o número de licenças emitidas, atendimentos realizados (incluindo animais silvestres) e o progresso em relação às metas anuais. Gráficos e indicadores facilitam a leitura e a tomada de decisões.
- **Gerenciamento Completo:** O sistema oferece funcionalidades robustas para o cadastro e controle de:
  - Metas anuais de licenças.
  - Licenças emitidas.
  - Atendimentos gerais.
  - Atendimentos a animais silvestres.
- Controle de Acesso: Gerencie o acesso ao sistema com um módulo de cadastro de usuários, garantindo que apenas pessoas autorizadas possam visualizar e manipular os dados.
